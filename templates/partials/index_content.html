Hello! This is an example page using Afflatus

A Flask-based web application with HTMX for dynamic frontend interactions and Pixeltable for data operations.

When in doubt, create another framework
  • Flask >=3.1.2
  • Flask-HTMX >=0.4.0
  • Pixeltable >=0.5.0
  • Python >=3.11
from flask import Flask, render_template
from flask_htmx import HTMX

app = Flask(__name__)
htmx = HTMX(app)

@app.route('/')
def index():
    return render_template('index.html')

if __name__ == '__main__':
    app.run(debug=True)
